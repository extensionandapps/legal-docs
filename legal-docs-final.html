5. Browser Permissions

Our extensions require certain browser permissions to function:

  • activeTab: To draw curtains on the current webpage or manage tab suspension

  • tabs: (Tab Freezer only) To identify and suspend inactive tabs

These permissions are used solely for the extension's core functionality and NOT for data collection.

6. Data Security

Since all data is stored locally on your device using your browser's secure storage API, your data security is managed by your browser's built-in security features. We recommend keeping your browser updated to the latest version.
